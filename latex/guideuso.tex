\hypertarget{guideuso_autotoc_md2}{}\doxysubsection{Como Navegar no Menu}\label{guideuso_autotoc_md2}
Para utilizar o menu, siga estas instruções\+:

{\bfseries{Insira a opção desejada\+:}} Digite o número correspondente à funcionalidade que deseja acessar e pressione Enter.

{\bfseries{Exemplo\+:}} Para calcular o logaritmo do vetor, insira {\bfseries{2}}.

{\bfseries{Siga as instruções\+:}} Caso uma funcionalidade exija entrada adicional (como um vetor ou matriz), forneça os dados solicitados.

{\bfseries{Voltar ao menu\+:}} Após executar uma funcionalidade, você será retornado automaticamente ao menu principal.

{\bfseries{Encerrar o programa\+:}} Escolha a opção {\bfseries{12}} para sair.\hypertarget{guideuso_autotoc_md3}{}\doxysubsection{Menu Principal}\label{guideuso_autotoc_md3}
\mbox{[}01\mbox{]} Construir matriz 4x18\+: Cria uma matriz 4x18 a partir de um vetor de 72 elementos.

\mbox{[}02\mbox{]} Logaritmo do vetor\+: Calcula o logaritmo base 10 de cada elemento do vetor (valores positivos).

\mbox{[}03\mbox{]} Mediana do vetor\+: Retorna o valor central de um vetor ordenado.

\mbox{[}04\mbox{]} Ordenar valores simétricos\+: Organiza números palíndromos do vetor em ordem crescente.

\mbox{[}05\mbox{]} Valores $>$ 2 divisíveis por 5\+: Filtra números maiores que 2 e múltiplos de 5.

\mbox{[}06\mbox{]} Soma das metades\+: Soma os elementos correspondentes da 1ª e 2ª metade do vetor (tamanho par).\hypertarget{guideuso_autotoc_md4}{}\doxysubsection{Funções Adicionais}\label{guideuso_autotoc_md4}
\mbox{[}07\mbox{]} Ajuda\+: Exibe este guia.

\mbox{[}08\mbox{]} Vetor baralhado\+: Embaralha os elementos do vetor.

\mbox{[}09\mbox{]} Decompor ímpares\+: Calcula os fatores primos de números ímpares no vetor.

\mbox{[}10\mbox{]} Matriz 18x18\+: Gera uma matriz 18x18 preenchida com valores.

\mbox{[}11\mbox{]} Transposta 18x18\+: Retorna a matriz transposta da 18x18 gerada.

\mbox{[}12\mbox{]} Sair\+: Encerra o programa.\hypertarget{guideuso_autotoc_md5}{}\doxysubsection{Dicas Adicionais}\label{guideuso_autotoc_md5}

\begin{DoxyItemize}
\item {\bfseries{Leia atentamente o menu\+:}} Certifique-\/se de entender o propósito de cada opção antes de selecioná-\/la.
\item {\bfseries{Preparação dos dados\+:}} Se o programa pedir um vetor ou matriz, forneça os dados no formato correto.
\item {\bfseries{Ajuda\+:}} Consulte a opção \mbox{[}07\mbox{]} para retornar a este guia sempre que necessário. 
\end{DoxyItemize}
